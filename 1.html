部署合约-1
在这篇文章里面我们来介绍如何使用基础工具实现合约的部署和调用，关于软件的安装，请自行搜索相关文档
需要使用到的软件
Ganache(): 提供一个可以本地运行的虚拟环境
Geth(1.10.8): 客户端，使用它来与链上交互
Solc(0.8.7): 编译写好的智能合约
Ubuntu(21.04)
首先，创建一个合约。这个合约的功能是可以创建一个口袋精灵，只要提供给_createPokemon函数俩个参数。使用vim编辑器或别的文本编辑器保存这个合约
// SPDX-License-Identifier: GPL-3.0 pragma solidity 0.8.7; contract PokemonFactory { // 一个结构体，有俩个字段，口袋精灵的名称和图鉴 struct Pokemon { string name; uint handbook; } // 一个数组 Pokemon[] public pokemons; // 函数，创建一个口袋精灵，需要俩个参数，_name和_handbook function _createPokemon(string memory _name, uint _handbook) public { // 在数组中加入实例化的结构体 pokemons.push(Pokemon(_name, _handbook)); } }
Go
编译合约
solc --optimize --combined-json abi,bin PokemonFactory.sol > PokemonFactory.json echo -n "pokemon=" > PokemonFactory.js cat PokemonFactory.json >> PokemonFactory.js
Go
